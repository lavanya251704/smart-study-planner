# smart-study-planner
A high-performance, dark-themed study dashboard for CS students. Features a 10-minute micro-sprint timer, priority-based task management, and dynamic visual progress tracking to eliminate procrastination and master complex coding tasks.

How It Works (The Efficiency Loop)

1.  Plan: Break your project into 10-minute "Micro-Tasks."
2.  Execute: Start the 10-minute timer. Eliminate all distractions.
3.  Track:Hit "Done" to watch your progress bar grow and your sprint count increase.
